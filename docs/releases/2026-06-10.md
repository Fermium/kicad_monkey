# 2026-06-10 Release

Release version: `2026.6.10`

This release fixes pin-name markup rendering in symbol SVG output and ships
refreshed test infrastructure for the public `kicad-monkey` package.

## Notes

- Fixes GitHub issue #1: `~{...}` overbar, `_{...}` subscript, and `^{...}`
  superscript in pin names are now parsed and rendered instead of drawn
  literally. Verified against `kicad-cli sym export svg` for both the KiCad
  stroke font and TTF-faced pin fonts.
- Default symbol SVG theme now matches KiCad CLI output (body fill, outline
  stroke widths, pin-number color); custom theme overrides are unchanged.
- Test corpus archive refreshed with overbar markup fixtures and a cleaned
  packaging policy that separates regenerable runtime products from genuine
  editor debris.
- Staged KiCad CLI oracle builds are now pinned in
  `tools/kicad-cli/MANIFEST.toml` for deterministic reference resolution.
