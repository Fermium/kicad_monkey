# 2026-05-31 Release

Release version: `2026.5.31`

This is the first public testing release for `kicad-monkey` as a standalone
library package. It is intended to prove the package boundary with
`bom_cruncher`, `lib_cruncher`, and the first `kicad-cruncher` integration.
Minor follow-up releases are expected as those integrations harden the public
API contract.

Release scope:

- date-version package metadata and changelog coverage;
- promoted package-root API contract for parser, OOP facade, project/design,
  IR/SVG, and utility exports;
- design-doc ownership for promoted public classes and major interfaces;
- package-local public KiCad corpus archive at `tests/corpus/kicad.zip`;
- Rack L0 and L99 public release gates;
- package-wide pyright at zero diagnostics under the public release signoff;
- GitHub CI, contribution, issue, and release workflow metadata.

Known follow-up work:

- finish human review of the archived KiCad corpus before public push;
- keep package pyright at zero while internal tools consume the release API;
- use the first `kicad-cruncher` integration to promote or retire provisional
  `__all__` exports.
