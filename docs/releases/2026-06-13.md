# 2026-06-13 Release

Release version: `2026.6.13`

This release hardens KiCad netlist parity for hierarchical projects and ships a
sanitized real-world backplane fixture that exercises the regression.

## Notes

- Matches KiCad CLI netlist structure for hierarchy cases where a sheet is not
  on the board: child components are excluded while parent-side sheet-pin nets
  remain connected.
- Aligns component metadata export with KiCad's S-expression netlist output,
  including shown field text, blank field handling, libsource, sheetpath,
  tstamps, properties, and units blocks.
- Improves multi-unit symbol handling for unit names, common pins, stacked pin
  expansion, unit-specific metadata, and duplicate reference suppression.
- Adds the sanitized `4-ch-backplane` project to the packed corpus as an active
  real-world fixture for netlist, schematic SVG/IR, and PCB SVG/IR tests.
- Keeps remaining one-pin unconnected-net naming and metadata-order drifts as
  explicit expected failures so CI continues to catch new structural netlist
  regressions.
